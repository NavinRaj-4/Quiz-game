# 🧠 Python Quiz Game

A simple command-line quiz game built with Python.

## Features

- Asks multiple questions
- Checks user answers
- Tracks score
- Displays final result

## How to Run

```bash
python quiz.py
```

## Concepts Used

- Dictionaries
- Loops
- Conditional statements
- User input
- String methods
